LangGraph Agent Flow

Week 7 — Base Graph (7 nodes): START → guardrail → [in-scope?] ├─ NO → out_of_scope → END └─ YES → retrieve → tool_retrieve → grade_documents ├─ relevant → generate_answer → END └─ irrelevant → rewrite_query → retrieve (max 2) Week 8 — Extended Graph (11 nodes): START → guardrail → [in-scope?] ├─ NO → out_of_scope → END └─ YES → retrieve → tool_retrieve → grade_documents ├─ irrelevant → rewrite_query → retrieve (max 2) └─ relevant → synthesis_node → contradiction_node → generate_answer → critic_node → [quality ≥ 0.7?] ├─ YES → memory_node → END └─ NO → generate_answer (max 2 retries)
Synthesis
Groups chunks by paper, finds themes + agreements + gaps
Contradiction
Detects conflicting findings, methods, conclusions across papers
Critic
Scores faithfulness, completeness, citations; retries if < 0.7
Memory
Retrieves prior research + stores new findings (Redis, TTL 90d)
